This is the official landing page and documentation hub for **Eidos**, an agentic image editor designed for precise control and creative experimentation.

## Project Overview

The `labwebsite` serves as the public face of the Eidos project, demonstrating its capabilities and aesthetic philosophy. It is built with:

- **React 19**: For modern UI components.
- **Tailwind CSS v4**: For utility-first styling.
- **Vite**: For fast development and building.
- **Framer Motion**: For complex animations.

## Key Features

- **Adaptive Agentic System Visualization**: Demonstrates the multi-agent workflow.
- **Semantic Control Showcase**: Highlights the granular editing capabilities.
- **Waitlist Integration**: Captures user interest using Netlify Forms.

## Getting Started

### Prerequisites

- Node.js >= 18
- pnpm >= 8

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

### Build

```bash
pnpm build
```
